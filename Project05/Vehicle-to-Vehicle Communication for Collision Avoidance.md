# Step 3 — LSTM Matches the Problem Naturally

Why LSTM becomes popular here:

Because traffic is sequential.

Vehicle behavior depends on:

* previous movement
* previous velocity
* previous lane behavior
* previous acceleration

Meaning:
past affects future.

That is exactly what LSTM was designed for.

Sequence learning.

So the model choice itself feels logical.

You are not forcing AI artificially.

# Understanding the Core Idea Deeply

Suppose a vehicle behaves like this:

| Time | Speed | Lane | Distance to Front Car |
| ---- | ----- | ---- | --------------------- |
| t1   | 50    | Left | 40m                   |
| t2   | 52    | Left | 35m                   |
| t3   | 54    | Left | 28m                   |
| t4   | 56    | Left | 18m                   |

The system learns:

“distance is reducing dangerously while speed increases.”

Then:
collision probability rises.

That is fundamentally:
pattern recognition across time.

Which becomes:
sequence prediction.

Which becomes:
LSTM learning.

So suddenly:
the theory becomes connected to reality.
